---
description: Guarda un commit de lo que hicimos hoy
---

Este workflow automatiza el guardado local de los cambios realizados en la sesión actual.

1. Preparar archivos para el commit:
// turbo
`git add .`

2. Crear el commit con un mensaje descriptivo:
// turbo
`git commit -m "📦 Refactor: Consolidación de workflows, rediseño de rejilla POS y ajustes de OTP en WhatsApp (v4.6.9)"`

3. Confirmar que el historial esté limpio:
// turbo
`git status`
